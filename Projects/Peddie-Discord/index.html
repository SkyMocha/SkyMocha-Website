Peddie Discord

For Peddie Uses Only

During the creation of a Peddie School only Discord server, I realized that there would need to be some way of verifying students to make sure they were students. To solve this problem, I made a Discord bot that through Google authentication on a localhost server checked to see if their email contained the peddie.org extension. As a bonus, since all student emails have the extension -xx for the graduation year, I was able to automatically assign roles for students depending on their graduation year.


!verify

Sends the User a glitch.io link to a Google verification page
