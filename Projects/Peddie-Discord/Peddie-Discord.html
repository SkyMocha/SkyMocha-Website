Peddie Discord

For Peddie Uses Only

To solve the problem of security on a school-only Discord, I implemented a Discord Bot. The bot works of Google authentication, checking to see if the user has a peddie.org domain for their email. Users who pass the verification will recieve a role on the server that gives them access to all channels.


!verify

Sends the User a glitch.io link to a Google verification page
