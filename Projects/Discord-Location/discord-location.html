Discord Location

See if your friends live near you!

"Location" is a Discord bot created by NickStudios. The bot's purpose is viewing your distance relative to your Discord friends! Your location is private, only relative distance is visible.

Uses a the MapQuest API connected to a MongoDB database working off user home-town coordinates.


*register Town, State/Province/Country

Registers a user in the location database


*change Town, State/Province/Country

Updates a user in the location database

*user @user

Gets the distance between you and the user
