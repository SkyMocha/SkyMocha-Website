PiPlant

First & last HS group project: A talking plant

Overview

A talking plant using a Python flask server connected to a Raspberry Pi feeding realtime into web app.
Group project with one junior and another senior Fall term of senior year.
My job primarily focuses on the web app & data visualizations

Back-end

The backend uses a raspberry pi using a external microphone, external speaker, soil humidity sensor, and finally a temperature sensor.
The primary person in charge of the backend was Rohan '22 Although I helped out with certain aspects of the Raspberry Pi itself as I worked extensively with one for my nickPosr project. Finally, I put in place certain parts of the chat-bot element, as I was more familiar with creating custom data structures for that purpose due to my time with Discord Bots.

Front-end

I was fully in-charge of the front-end through the use of a stub server/API I created. The front-end uses d3.js as a library to help with data visualizations. The visualizations are updated realtime as new data is introduced from the back-end.
